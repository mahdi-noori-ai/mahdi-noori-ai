## 🧪 Selected Project Directions

> Repos are in progress / being refactored into clean, production- and research-style projects.

- ⚙️ **Dev & Automation Toolkit for ML/LLM**  
  A set of fast-start templates and CLI tools to create new ML/LLM projects with one command (data.py, model.py, run.py + cfg.json + metrics + cards), including automation for common tasks like dataset loading, experiment logging, and report generation.

- 🧠 **LLM Agents & Workflow Orchestration**  
  Agent-style systems that combine LLMs with external tools (APIs, search, code runners, vector DBs) to perform multi-step tasks autonomously, with careful logging, guardrails, and evaluation.

- 🔎 **Local RAG Studio**  
  A private, CPU-friendly RAG stack (FAISS + HF + simple UI) where you can plug in your own datasets, run experiments, and evaluate with structured QA metrics and qualitative failure analysis.

- 🔤 **NLP / Text Intelligence Lab**  
  A collection of well-structured NLP projects (classification, NER, sentiment, topic modeling) with shared infrastructure: common data pipeline, shared evaluation code, and clear baseline vs. improved models.

- 🧩 **GenAI for Productivity & Prototyping**  
  Small, focused GenAI demos: prompt libraries, lightweight fine-tuning experiments, and UI prototypes (e.g., Streamlit/FastAPI frontends) that show how LLMs can accelerate real workflows (analysis, coding, documentation, research).
